Installing Lakka to the drive

Downloading Windows image flasher utility

On Windows, you need a graphical tool to flash Lakka to your SD Card or USB drive.

This tool is called Etcher, and it is free.

<%= render 'partials/dl-button', :url => 'https://etcher.io/', :detail => 'for Windows ~ 82MB', :what => 'Etcher' %>

Flash the image

Run Etcher.

Select the Lakka image and select the drive, and hit the Flash button.
