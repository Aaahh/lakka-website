Download and unzip Lakka

This step is independant of your OS choice, the downloaded image can be used on any OS.

However, the image is specific to the target hardware, as its name should reflect it.

<%= render 'partials/dl-button', :url => 'https://github.com/lakkatv/Lakka/releases/download/2014-05-05/Lakka-RPi.arm-devel-20140515211630-r18096-g86a5168.img.gz', :detail => "for Raspberry Pi ~ 74MB" %>

Please note that this stable version is 6 month old, if you need Raspberry Pi B+ support, or want to see the nice graphical menu, get a nightly build from here instead.

On Windows, please use 7zip to unzip Lakka.
