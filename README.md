# Lakka news Wadsite

## Synopsis

This project is the news website for Lakka

## Development environment

This website is based on nanoc, a static generator CMS.

Requirement:

- Ruby: 2.0.0 or newer
- Gems:
  - nanoc
  - kramdown
  - adsf (optionnal but useful)
